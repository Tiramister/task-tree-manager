# Task Tree Manager

タスクを木構造で管理する Web アプリケーション。データはブラウザの localStorage に保存されます。

## 機能

- **ツリー形式のタスク管理** - タスクに親子関係を持たせ、階層構造で表示・管理
- **タスクの作成・編集・削除** - タイトル、ステータス、説明、期限、作業メモを記録
- **折り畳み表示** - サブタスクの展開・折り畳み（一括操作も可能）
- **完了履歴** - 完了したタスクを日付ごとにまとめて振り返り
- **レスポンシブ対応** - PC・スマートフォンどちらでも利用可能

## 開発

```bash
npm install
npm run dev   # ローカル実行
```
