## ADDED Requirements

### Requirement: メニューボタンにアイコンを表示する
エクスポートボタンとインポートボタンのテキスト左側に、それぞれ対応するアイコン画像を表示しなければならない（SHALL）。

#### Scenario: エクスポートボタンにアイコンが表示される
- **WHEN** ユーザーがハンバーガーメニューを開く
- **THEN** エクスポートボタンのテキスト左側に `/export.png` のアイコンが表示される

#### Scenario: インポートボタンにアイコンが表示される
- **WHEN** ユーザーがハンバーガーメニューを開く
- **THEN** インポートボタンのテキスト左側に `/import.png` のアイコンが表示される
