## Context

右上の3点メニュー（`src/App.tsx`）にはエクスポートとインポートの2項目がある。ここに GitHub リポジトリへの外部リンクを追加する。

## Goals / Non-Goals

**Goals:**
- 3点メニューに GitHub リンクを追加し、新しいタブでリポジトリを開く

**Non-Goals:**
- メニューの構造やスタイルの変更
- 他の外部リンクの追加

## Decisions

- **アイコン**: `public/github.svg` を `<img>` タグで表示する。既存のエクスポート・インポートアイコンと同じ方式。
- **リンクの開き方**: `window.open` で `_blank` ターゲットを指定し、新しいタブで開く。`<a>` タグではなくボタン + `window.open` とすることで、既存メニュー項目のスタイルと統一する。
- **配置位置**: インポートの下（メニュー末尾）に配置する。

## Risks / Trade-offs

- リスクは特にない。既存のメニュー項目と同じパターンでの追加であり、影響範囲は最小限。
