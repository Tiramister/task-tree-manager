## MODIFIED Requirements

### Requirement: ユーザー名クリック時のドロップダウンメニュー
ログイン済み時にヘッダー右上のユーザー名をクリックすると、ドロップダウンメニューが表示されなければならない（SHALL）。メニューには他のログイン済みユーザー一覧、「別のアカウントでログイン」、および「ログアウト」項目を含まなければならない（SHALL）。

#### Scenario: ドロップダウンメニューの表示（複数ユーザーログイン済み）
- **WHEN** 複数ユーザーがログイン済み状態でヘッダー右上のユーザー名をクリックする
- **THEN** 他のログイン済みユーザー一覧、「別のアカウントでログイン」、「ログアウト」を含むドロップダウンメニューが表示される

#### Scenario: ドロップダウンメニューの表示（単一ユーザーログイン済み）
- **WHEN** 1ユーザーのみログイン済み状態でヘッダー右上のユーザー名をクリックする
- **THEN** 「別のアカウントでログイン」と「ログアウト」を含むドロップダウンメニューが表示される

#### Scenario: ログアウトの実行
- **WHEN** ドロップダウンメニューの「ログアウト」をクリックする
- **THEN** ログアウトが実行され、他にセッションがあれば自動切り替え、なければヘッダー右上が「ログイン」ボタンに戻る

## ADDED Requirements

### Requirement: ユーザー切り替えメニュー項目
ドロップダウンメニューには現在のユーザー以外のログイン済みユーザーがリスト表示されなければならない（SHALL）。各ユーザー名をクリックするとそのユーザーに切り替わらなければならない（SHALL）。

#### Scenario: 他ユーザーへの切り替え
- **WHEN** ドロップダウンメニューで他のログイン済みユーザー名をクリックする
- **THEN** そのユーザーに切り替わり、ヘッダー右上に新しいユーザー名が表示される

#### Scenario: 切り替え後のメニュー
- **WHEN** ユーザーを切り替えた後にドロップダウンメニューを開く
- **THEN** 切り替え前のユーザーがリストに表示される

### Requirement: 別アカウントでログインオプション
ドロップダウンメニューには「別のアカウントでログイン」オプションを含まなければならない（SHALL）。クリックするとログインダイアログが表示されなければならない（SHALL）。

#### Scenario: 別アカウントでログインダイアログ表示
- **WHEN** ドロップダウンメニューの「別のアカウントでログイン」をクリックする
- **THEN** ログインダイアログが表示される

#### Scenario: 別アカウントでのログイン成功
- **WHEN** 「別のアカウントでログイン」からログインダイアログを開き、新しいアカウントでログインする
- **THEN** 新しいアカウントがセッション一覧に追加され、そのアカウントがアクティブになる

### Requirement: 現在のユーザーの視覚的区別
ドロップダウンメニューで現在アクティブなユーザーは視覚的に区別されなければならない（SHALL）。チェックマークまたはハイライトで表示しなければならない（SHALL）。

#### Scenario: アクティブユーザーの表示
- **WHEN** ドロップダウンメニューを開く
- **THEN** 現在のアクティブユーザーにチェックマークまたはハイライトが表示される
