## MODIFIED Requirements

### Requirement: 子孫タスクの作業記録を表示

各子孫タスクは折り畳み可能で、展開すると詳細説明（description）と作業記録（notes）を表示しなければならない（SHALL）。詳細説明と作業記録の両方が存在する場合、間に空行を挟んで連続表示しなければならない（SHALL）。片方のみの場合はその内容だけを表示しなければならない（SHALL）。ラベル（「詳細説明:」等）は付けず、本文をそのまま表示しなければならない（SHALL）。テキストは、長い単語やURLを含む場合でもコンテナ内で折り返されなければならない（SHALL）。

#### Scenario: タスクを展開して詳細説明と作業記録を表示
- **WHEN** ユーザーが詳細説明と作業記録の両方を持つ子孫タスクを展開する
- **THEN** 詳細説明が先に表示され、空行を挟んで作業記録が続けて表示される

#### Scenario: 詳細説明のみの場合
- **WHEN** ユーザーが詳細説明のみを持つ子孫タスクを展開する
- **THEN** 詳細説明のみが表示される

#### Scenario: 作業記録のみの場合
- **WHEN** ユーザーが作業記録のみを持つ子孫タスクを展開する
- **THEN** 作業記録のみが表示される

#### Scenario: 両方未設定の場合
- **WHEN** ユーザーが詳細説明も作業記録も持たない子孫タスクを展開する
- **THEN** 「作業記録なし」の空状態メッセージが表示される

#### Scenario: 長いテキストの折り返し
- **WHEN** 子孫タスクの詳細説明または作業記録に折り返し不可能な長い文字列（URL等）が含まれる
- **THEN** テキストはコンテナの幅で強制的に折り返され、はみ出さない
