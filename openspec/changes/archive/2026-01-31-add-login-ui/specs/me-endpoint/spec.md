## ADDED Requirements

### Requirement: GET /me エンドポイント
サーバーは `GET /me` エンドポイントを提供しなければならない（SHALL）。認証済みセッションに紐づくユーザー名を JSON 形式で返す。レスポンスボディは `{ "username": "<ユーザー名>" }` の形式でなければならない（SHALL）。

#### Scenario: 認証済みユーザーの情報取得
- **WHEN** 有効なセッション Cookie を持つクライアントが `GET /me` にリクエストを送信する
- **THEN** ステータスコード 200 が返り、レスポンスボディに `{ "username": "<ユーザー名>" }` が含まれる

#### Scenario: 未認証ユーザーの情報取得
- **WHEN** セッション Cookie を持たないクライアントが `GET /me` にリクエストを送信する
- **THEN** ステータスコード 401 が返る（認証ミドルウェアにより）
