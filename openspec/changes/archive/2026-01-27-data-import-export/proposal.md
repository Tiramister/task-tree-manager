## Why

現在、タスクデータは localStorage にのみ保存されており、デバイス間での移行やバックアップの手段がない。ファイルを介したインポート・エクスポート機能を追加することで、データのバックアップ・復元・移行を可能にする。

## What Changes

- タスクデータを JSON ファイルとしてエクスポートする機能を追加
- JSON ファイルからタスクデータをインポートする機能を追加
- インポート時に既存データを置換するか確認する UI を追加

## Capabilities

### New Capabilities

- `data-export`: タスクデータを JSON ファイルとしてダウンロードする機能
- `data-import`: JSON ファイルを読み込んでタスクデータを復元する機能

### Modified Capabilities

（なし）

## Impact

- **コード**: タスクストアにインポート・エクスポート用のアクションを追加
- **UI**: インポート・エクスポートを実行するための UI 要素を追加
- **依存関係**: 追加の外部ライブラリは不要（ブラウザの File API を使用）
