## ADDED Requirements

### Requirement: タスクデータを JSON ファイルとしてエクスポートできる

ユーザーがエクスポートを実行すると、現在のすべてのタスクデータを含む JSON ファイルがダウンロードされなければならない（SHALL）。

#### Scenario: タスクが存在する状態でエクスポートする
- **WHEN** タスクが1つ以上存在する状態でエクスポートを実行する
- **THEN** すべてのタスクデータを含む JSON ファイルがダウンロードされる

#### Scenario: タスクが存在しない状態でエクスポートする
- **WHEN** タスクが1つも存在しない状態でエクスポートを実行する
- **THEN** 空の配列を含む JSON ファイルがダウンロードされる

### Requirement: エクスポートファイルはタスクの全フィールドを含む

エクスポートされた JSON ファイルには、各タスクの全フィールド（id, title, status, createdAt, および設定済みのオプショナルフィールド）が含まれなければならない（SHALL）。

#### Scenario: オプショナルフィールドを持つタスクをエクスポートする
- **WHEN** description, parentId などのオプショナルフィールドが設定されたタスクをエクスポートする
- **THEN** エクスポートファイルにそれらのオプショナルフィールドも含まれる

### Requirement: エクスポートファイル名は日時を含む

ダウンロードされるファイル名は `tasks-YYYYMMDD-HHmmss.json` 形式でなければならない（SHALL）。

#### Scenario: エクスポート時のファイル名
- **WHEN** 2025年1月15日 14:30:00 にエクスポートを実行する
- **THEN** ファイル名が `tasks-20250115-143000.json` となる
