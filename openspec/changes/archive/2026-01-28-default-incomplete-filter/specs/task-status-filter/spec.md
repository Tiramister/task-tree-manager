## MODIFIED Requirements

### Requirement: 未完了タスクフィルタの切り替え
一覧画面にトグルボタンを設置し、ユーザーが完了タスクの表示/非表示をワンボタンで切り替えられなければならない（SHALL）。デフォルトではフィルタは有効（未完了のみ表示）とする。

#### Scenario: フィルタを有効にする
- **WHEN** ユーザーがフィルタボタンを押下する（フィルタ無効状態）
- **THEN** ステータスが `completed` のタスクが一覧から非表示になる

#### Scenario: フィルタを無効にする
- **WHEN** ユーザーがフィルタボタンを再度押下する（フィルタ有効状態）
- **THEN** すべてのタスクが一覧に表示される

#### Scenario: 初期表示時のフィルタ状態
- **WHEN** ユーザーがタスク一覧画面を開く
- **THEN** フィルタが有効な状態（未完了のみ表示）で表示される
