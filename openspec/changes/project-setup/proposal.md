## Why

タスクツリーマネージャーを開発するための基盤が必要。React + TypeScript + Vite でモダンなフロントエンド環境を構築し、Tailwind CSS と Shadcn/ui でUIを効率的に開発できる状態にする。

## What Changes

- Vite + React + TypeScript プロジェクトの初期化
- Tailwind CSS のセットアップ
- Shadcn/ui のセットアップ
- 基本的なディレクトリ構造の作成
