## ADDED Requirements

### Requirement: 空状態の表示
タスクが0件の場合、ツリービュー領域にガイドメッセージとタスク作成ボタンを表示しなければならない（SHALL）。

#### Scenario: タスクが0件
- **WHEN** すべてのタスクが削除され、タスクが0件になる
- **THEN** 「タスクがありません」等のメッセージと新規タスク作成ボタンが表示される

#### Scenario: タスク作成後
- **WHEN** 空状態からタスクを作成する
- **THEN** ガイドメッセージが消え、通常のツリービューが表示される

### Requirement: UI の視覚的改善
アプリケーション全体の見た目を整え、使い勝手を向上させなければならない（SHALL）。ヘッダー、余白、色使い等の微調整を含む。

#### Scenario: アプリの外観
- **WHEN** ユーザーがアプリを開く
- **THEN** 整ったレイアウトと視覚的に統一されたデザインが表示される
