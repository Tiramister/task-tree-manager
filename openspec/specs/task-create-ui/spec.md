## Requirements
### Requirement: ルートタスク作成ボタン

ツリービューの上部に「新規タスク」ボタンを表示しなければならない（SHALL）。ボタンをクリックするとタスク作成ダイアログが開く。

#### Scenario: ルートタスクを作成する

- **WHEN** ユーザーが「新規タスク」ボタンをクリックする
- **THEN** タスク作成ダイアログが開く

### Requirement: 子タスク作成メニュー

各タスクアイテムにアクションメニュー（三点リーダーアイコン）を表示しなければならない（SHALL）。メニューから「子タスクを追加」を選択できる。

#### Scenario: 子タスクを作成する

- **WHEN** ユーザーがタスクの三点リーダーをクリックし、「子タスクを追加」を選択する
- **THEN** 親タスクIDが設定された状態でタスク作成ダイアログが開く

### Requirement: タスク作成ダイアログ

タスク作成ダイアログはタイトル入力フィールドと作成ボタンを持たなければならない（SHALL）。

#### Scenario: タイトルを入力してタスクを作成する

- **WHEN** ユーザーがタイトルを入力し、「作成」ボタンをクリックする
- **THEN** 新しいタスクが作成され、ツリービューに表示される

#### Scenario: 空のタイトルでは作成できない

- **WHEN** ユーザーがタイトルを空のまま「作成」ボタンをクリックする
- **THEN** タスクは作成されず、ダイアログは開いたまま

#### Scenario: キャンセルする

- **WHEN** ユーザーが「キャンセル」ボタンをクリックする
- **THEN** ダイアログが閉じ、タスクは作成されない
